# 📋 MokoPaste Feedback

Welcome to the MokoPaste community! Use this repository to report bugs or suggest new features.

## 🚀 Useful Links
- **Official Website**: [mokopaste.com](https://mokopaste.com)
- **Contact Support**: [support@mokopaste.com](mailto:support@mokopaste.com)

## 🐛 How to Report a Bug
1. Check the [Existing Issues](https://github.com/mokopaste/feedback/issues) to see if it's already reported.
2. If not, click **New Issue** and use our template.

## ✨ Feature Requests
Have an idea to make MokoPaste better? We'd love to hear it! Open an issue with the tag `enhancement`.
